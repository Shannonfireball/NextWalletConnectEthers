## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Click on the connect button.

Once your wallet is connected 
Add the contract address and the abi to the code and call the function you want.
Make sure to add a deployed contract or you can run one on local using anvil.


You can start editing the page by modifying `app/Layout.tsx`.
